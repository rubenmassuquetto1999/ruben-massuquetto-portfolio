# 🚀 Portfolio Pessoal | Ruben Massuquetto

Bem-vindo ao repositório do meu estúdio digital. Este projeto é a materialização da minha transição para o desenvolvimento Front-End, onde aplico o rigor técnico do planejamento industrial (PCP) e a visão analítica da Ciência de Dados para construir interfaces robustas, lógicas e de alta performance.

🔗 **Visualizar Projeto:** [rubenmassuquetto.com](https://rubenmassuquetto.com)

---

## 🛠 Tecnologias e Ferramentas

O projeto foi desenvolvido focando em **Vanilla Web Stack**, priorizando performance e controle total sobre o código:

- **Linguagens:** HTML5, CSS3, JavaScript (ES6+)
- **APIs Integradas:**
    - **GitHub API:** Consumo dinâmico de repositórios para automação de vitrine de projetos.
    - **Microlink API:** Geração automática de thumbnails (screenshots) em tempo real.
- **Serviço de Mensageria:** Formspree para tratamento de formulários AJAX.
- **Servidor:** Apache com configuração de Clean URLs (.htaccess).

---

## 🎯 Diferenciais Técnicos

### ⚡ Performance & Core Web Vitals
O site foi otimizado para atingir scores máximos no **Google PageSpeed Insights**.
- Imagens otimizadas com `aspect-ratio` para evitar CLS (Cumulative Layout Shift).
- Carregamento de scripts não bloqueantes (`defer`).
- Uso de propriedades CSS eficientes para transições e animações de interface.

### 📐 Engenharia Visual & UX
- **Design Responsivo:** Layout elástico adaptado para Mobile (Portrait/Landscape), Tablets e Desktops.
- **Clean URLs:** Implementação de rotas amigáveis (ex: `/projetos` em vez de `/projetos.html`) para melhor SEO e experiência do usuário.
- **Filtro em Tempo Real:** Sistema de busca dinâmica na seção de projetos desenvolvido com Vanilla JS.

### 📈 Foco em Conversão (Growth)
- Botão flutuante de WhatsApp com mensagem pré-preenchida.
- Formulário de contato blindado com feedback de status em tempo real.
- Conformidade com LGPD na coleta de dados básicos de contato.
